# Resume Project — SQL Analytics + Dashboard

Load a real public dataset into PostgreSQL, answer pointed business questions with SQL, and present the findings in a dashboard. No software-engineering background required — ideal for analyst roles and career-changers.

**📘 Full step-by-step guide:** https://alignresume.com/projects/sql-analytics-dashboard/

> Build it under **your own GitHub account**, committing each question's query + finding.

## Stack
SQL · PostgreSQL · a dashboard tool (Metabase / Power BI / Tableau Public)

## Suggested structure
```
load.py / load.sql     # load the dataset into Postgres
queries/               # one well-commented .sql per business question
dashboard/             # screenshots / export of your dashboard
README.md              # the questions + your findings narrative
```

## Build it (milestones — commit each)
- [ ] Pick a real dataset + write down 5–8 business questions
- [ ] Load it into PostgreSQL; document the schema
- [ ] Clean it (nulls, types, duplicates) and note what you changed
- [ ] One commented query per question (joins, aggregations, a window function)
- [ ] Build a dashboard (trend, breakdown, top-N) and commit screenshots
- [ ] Write a "findings" narrative: asked / found / next
